1. Mic records continuously as raw PCM.

2. Silero VAD listens for speech.

3. When speech starts:
   start buffering audio.

4. When silence lasts ~800 ms:
   stop buffering.

5. Save the captured utterance:
   /tmp/pi-voice/utterance.wav

6. Ensure audio is:
   16 kHz
   mono
   WAV container
   ideally PCM; runner converts to float32 [-1, 1] internally if supported.

7. Send utterance.wav to Gemma 4 E2B with prompt:
   "Transcribe this speech. Output only the text."

8. Gemma returns:
   "pi check the server logs"

9. Filter layer:
   starts with "pi" → strip prefix
   result: "check the server logs"

10. Inject into Pi:
   pi.sendUserMessage("check the server logs")
